## ğŸ¨ Visual Design Details

### Color Usage
- **Background**: White/Gray-50 alternating sections
- **Headers**: Gray-900 (almost black)
- **Body Text**: Gray-600
- **Purple**: Logo, links, accents
- **Green**: CTAs, success states
- **White**: Cards, overlays

### Typography Hierarchy
```
Hero H1: 56px/64px, Bold
Section H2: 48px/56px, Bold  
Feature H3: 32px/40px, Semibold
Body: 18px/28px, Regular
Small: 14px/20px, Regular
```

### Shadow System
```scss
shadow-sm: 0 1px 2px rgba(0,0,0,0.05)
shadow: 0 4px 6px rgba(0,0,0,0.07)  
shadow-lg: 0 10px 15px rgba(0,0,0,0.1)
shadow-xl: 0 20px 25px rgba(0,0,0,0.1)
```

### Animation Strategy
1. **Hero**: Fade in sequence on load
2. **Stats**: Count up when in view
3. **Features**: Slide in from sides
4. **Cards**: Hover lift effect
5. **CTAs**: Pulse/glow animation
