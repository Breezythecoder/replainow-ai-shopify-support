### SPA Routing Strategy:
**Problem:** GitHub Pages doesn't support client-side routing  
**Solution:** 404.html = copy of index.html

**How it works:**
1. User visits `/privacy`
2. GitHub Pages: 404 (no privacy.html file)
3. Serves `404.html` (which is copy of `index.html`)
4. Full React app loads
5. React Router reads URL pathname
6. Routes to PrivacyPolicy component
7. Page displays correctly âœ…

**Why this works:**
- No redirect loops (old problem!)
- Clean URLs (no .html extension)
- React Router handles everything
- Simple & reliable
